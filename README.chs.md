<h1 align="center">CherryGrove</h1>

脚本驱动的可高度自定义的方块游戏框架。

## 特性（目标）

1. 内核无游戏功能，几乎所有游戏元素都可以自定义。
2. 一套完整的“模组”API，接受 JSON、JavaScript、WebAssembly 和 Luau（Lua 的带类型变体）。
3. 通过内置可用组件实现高性能功能添加，内容包开发者不必重复发明轮子。
4. 拥有兼容许多平台（Windows、Linux、Mac、Android、iOS、Web 浏览器 + emscripten）的能力和计划。
5. 对非商业使用免费，且源码可用。
