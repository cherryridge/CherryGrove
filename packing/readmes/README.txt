Folder Introduction:
assets: Stores assets necessary for CherryGrove to run, including fonts and logos. Not for adding packs. Don't modify or CherryGrove may not be able to launch.
cache: Stores cache files generated by CherryGrove. Can be deleted to free up disk space occasionally when CherryGrove is closed.
captures: Stores screenshots and screen recordings.
logs: Stores logs of CherryGrove. Maybe useful for reporting bugs.
packs: Stores content packs. Here is the place to add packs to CherryGrove, in the format of folder or archive. Ensure that manifest.json of the pack is one folder away from packs folder.
saves: Stores CherryGrove worlds.

File Introduction:
CherryGrove.exe: The CherryGrove program.
settings.json: The settings storage for CherryGrove. Modify with knowledge and caution.
unins000.dat, unins000.exe: Program and data for uninstalling CherryGrove. Doubleclick to uninstall. Don't modify.
README.txt: This document.
LICENSE.txt: The Terms of Use of CherryGrove.
